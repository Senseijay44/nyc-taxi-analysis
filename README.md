# DuckDB NYC Taxi Exploration

## Objective
Explore DuckDB as a local analytical engine using the NYC taxi dataset.

### Key Steps
- Multi-file CSV ingestion
- Local DuckDB database creation
- Data profiling and anomaly detection
- Clean analytical view
- Performance comparison (CSV vs DuckDB vs Parquet)

### Dataset
NYC Yellow Taxi trip data (2015–2016)

### Key Takeaway
Modern columnar analytics engines can analyze millions of rows locally with minimal infrastructure.
